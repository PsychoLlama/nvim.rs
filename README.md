# nvim.rs

A rewrite of Neovim [v0.12.4](https://github.com/neovim/neovim/releases/tag/v0.12.4) to Rust.

## Purpose

Two goals:

1. Evaluate different coding agents against a complex project, specifically one from its training data.
2. Court the profane: integrate ideas from Emacs directly into Neovim.

## Caveats

- Rust was generated by [c2rust](https://github.com/PsychoLlama/c2rust). It's unidiomatic and littered with `unsafe`.
- Future changes will be driven by coding agents. Quality only matters if it breaks my own workflow.
- No compatibility. Features may break or get removed according to my inscrutable whims.
- Continued development is dependent on me not getting bored.

## Usage

**Nix**

```bash
nix run 'github:PsychoLlama/nvim.rs#'
```

Only supported on Linux. Binary releases coming... someday.
